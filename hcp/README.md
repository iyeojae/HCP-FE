HCP (동아리 지원 서비스)
📌 프로젝트 소개

HCP는 동아리 지원 과정을 보다 쉽고 효율적으로 관리할 수 있도록 돕는
동아리 지원 서비스입니다.
지원자와 운영진 모두에게 직관적인 UI와 편리한 기능을 제공하는 것을 목표로 합니다.

🛠️ 기술 스택

Frontend: 리액트

Backend: 자바, 스프링

🎨 프론트엔드 진행 상황

로그인 페이지: 구현 완료 (보완 작업 진행 중)

메인 페이지: 구현 진행 중 - 사이드 탭 구현 완료

메뉴바,헤더: 구현 완료

마이 페이지, 목록 페이지 진행중

⏳ 개발 일정

프로젝트 기한 내 완성 목표

현재 진행 상황을 고려했을 때, 기한 내 완성은 힘들 것으로 판단됨

핵심 기능 우선 개발 후 점진적 개선 예정

(팀원 및 역할 작성 예정)
